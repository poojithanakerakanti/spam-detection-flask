# 📩 Spam Detection Web App

This is a simple Machine Learning web application built using Flask that detects whether a message is Spam or Not Spam.

## 🚀 Features
- Detects spam messages instantly
- Uses TF-IDF Vectorization
- Naive Bayes Machine Learning model
- Simple web interface
- Automatically opens in browser

## 🧠 Technologies Used
- Python
- Flask
- Scikit-learn
- HTML

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
python app.py

3. The app will open automatically in your browser.

## 📌 Example Inputs
- "Hello how are you" → Not Spam  
- "Congratulations you won lottery" → Spam  
- "Urgent! send your bank details" → Spam
